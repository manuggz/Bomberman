NivelMapa::ExtremoColision NivelMapa::colision ( SDL_Rect  rect,
int *  num_colisiones,
bool  soloBloquesNoTraspasables = false 
)

Detecta si uno de los extremos del rectangulo colisiona con un bloque con la propiedad de solido

Parameters
rectrectangulo
num_colisionesNumero de extremos en colision
soloBloquesNoTraspasables
Returns
El ultimo extremo detectado que colisiona

Nota, en el caso que sea un player el que quiera detectar la colision éste usa un cuadro de colision(size=10) menor que el de los tiles(size=16) por lo que num_colisiones será 1 o 2.

Se regresa i + 1 , para que el 0 represente NO COLISION, el i + 1 significa:

1 –> extremo TOP-LEFT 2 –> extremo TOP-RIGHT 3 –> extremo BOTTOM-RIGHT 4 –> extremo BOTTOM-LEFT

Si se quiere que solo sean los bloques que no se pueden traspasar
